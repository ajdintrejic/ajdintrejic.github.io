WorkLog - A simple Web App

July 7, 2020 1-minute read

Simple Web App built with React.js, Express and PostgreSQL

WorkLog is a web app which is made for logging workers data. User/worker can input data when and what has done in that time. It can also track location. Admins can look at aggregated data which can be useful for expanding their business or be more efficient at what they do.

The app is currently mostly done. It still needs tweaks most of which aren’t hard to implement. Things which need to be tweaked can be found here on the end of the blogpost.
